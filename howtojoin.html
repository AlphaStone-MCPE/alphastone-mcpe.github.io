How to Join

1

Launch Minecraft PE and click Play.

2

In the world selection screen click Edit.

3

Click External to add a new server.

4

Enter Server Details

Server Name: AlphaStone
Address: alphastone.ddns.net
Port: 13804
5

Click Add Server to save the server to list, then click on it!

Enjoy!
