Server Commands

SetHome

  • /sethome Set a home point at your current location.
  • /home Teleport back to your home point.

TimeVote

  • /vote <day/night> Start a vote to change the time in the world.
  • /vote <yes/no> Vote for or against a started time vote.

TeleportRequest

  • /tpa <player> Request to be teleported to another player.
  • /tphere <player> Request to teleport a player to your location.
  • /tpaccept Accept an incoming teleport request.
  • /tpdeny Deny an incoming teleport request.

BlockProtect

  • /bp lock Protect a block from being modified by others.
  • /bp unlock Remove protection from a block.
  • /bp add <player> Allow another player to interact with your protected block.
  • /bp remove <player> Remove a player's access to your protected block.
  • /bp info Get information about a protected block.

SimpleWarp

  • /addwarp <name> Add a new warp point. ADMIN ONLY
  • /delwarp <name> Delete an existing warp point. ADMIN ONLY
  • /warp <name> Teleport to a warp point.
  • /openwarp <name> Make a warp public (allow non-ops to use it).
  • /closewarp <name> Make a warp private (ops only).
